# Saleon — цифровой сотрудник

Лендинг «Saleon — продавец, который всегда на смене».

Статический сайт: `index.html` содержит шаблон `<x-dc>`, который на лету рендерит
рантайм `support.js` (подгружает React 18 и Babel с CDN и монтирует страницу).

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Точка входа (копия `Saleon.dc.html`). |
| `Saleon.dc.html` | Исходный экспорт из редактора. |
| `support.js` | Рантайм (`dc-runtime`), рендерит шаблон через React. |
| `uploads/` | Дизайн-документы, токены и CSS-переменные. |

## Локальный запуск

Нужен любой статический HTTP-сервер (открывать `file://` нельзя — React
подгружается по CDN с проверкой SRI):

```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

или

```bash
npx serve .
```

## Деплой

Проект — статика, разворачивается на Vercel без сборки (zero-config).
